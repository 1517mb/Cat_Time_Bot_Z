# 🐾 Cat Time Bot Z (FastAPI Edition)

Асинхронный Telegram-бот на базе **Aiogram 3** и **SQLAlchemy** для геймифицированного учета рабочего времени и выездов IT-специалистов. Бот не только считает отработанные часы, но и награждает пользователей достижениями, повышает уровни (от "Укротителя USB" до "Архитектора Систем") и рассылает ежедневные сводки (погода, валюты ЦБ РФ, крипта).

## ✨ Ключевые возможности
* ⏱ **Трекинг времени:** Команды `/join` и `/leave` для фиксации прибытия и ухода с точек.
* 🎮 **Геймификация:** Умная система начисления опыта, 100 уникальных IT-званий и система достижений ("Первая кровь" и др.).
* 📊 **Профиль:** Личный кабинет (`/profile`) с графическим прогресс-баром и статистикой.
* ⏰ **Фоновые задачи (APScheduler):** Настраиваемые уведомления прямо из чата.
* 🌤 **Ежедневные сводки:** * `/start_weather` — детальная погода, прогноз, фазы луны и магнитные бури (OpenWeather + NOAA + ephem).
  * `/start_currency` — курсы фиатных валют с динамикой роста/падения (API ЦБ РФ).
  * `/start_crypto` — курсы основных криптовалют (CoinGecko).

## 🛠 Стек технологий
* **Python 3.10+**
* **Фреймворк:** [Aiogram 3.x](https://docs.aiogram.dev/en/latest/)
* **База данных:** SQLite + [SQLAlchemy 2.0](https://docs.sqlalchemy.org/) (Асинхронная)
* **Планировщик:** [APScheduler](https://apscheduler.readthedocs.io/)
* **Сетевые запросы:** `aiohttp`

## 🚀 Инструкция по локальному запуску

### 1. Подготовка окружения
Склонируйте репозиторий (или откройте папку проекта) и создайте виртуальное окружение:

```bash
# Для Windows
python -m venv venv
venv\Scripts\activate

# Для Linux/macOS
python3 -m venv venv
source venv/bin/activate
```

### 2. Установка зависимостей

Установите необходимые библиотеки (если файла requirements.txt еще нет, установите вручную):

```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения

В корневой папке проекта создайте файл .env и добавьте туда ваши токены:

```bash

TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
OPENWEATHER_TOKEN=ваш_ключ_от_OpenWeatherMap
```

### 4. Инициализация базы данных и уровней
Перед первым запуском необходимо загрузить в базу данных иерархию уровней (титулов) и пороги опыта.
Убедитесь, что вы находитесь в папке cat_bot и выполните команду:
```bash
python -m scripts.init_level
```
Ожидаемый результат: сообщение об успешном добавлении 100 уровней в БД.

### 5. Запуск бота
```bash
python main.py
```
В консоли должно появиться сообщение Бот запущен!, после чего можно переходить в Telegram.

## 📂 Структура проекта

```plaintext
Cat_Time_Bot_Z/
├── cat_bot/
│   ├── core/               # Настройки БД, модели (models.py), запросы (crud.py)
│   ├── handlers/           # Обработчики команд (/profile, /join, /leave, расписания)
│   ├── services/           # Бизнес-логика (погода, валюты, геймификация, tasks.py)
│   └── scripts/            # Утилиты и скрипты миграции (init_levels.py)
├── main.py                 # Точка входа, инициализация бота и диспетчера
└── .env                    # Секретные ключи (не коммитить!)
```
